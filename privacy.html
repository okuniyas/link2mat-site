iCloud Sync

iCloud 同期

iCloud 동기화

Sincronização iCloud

Sincronización iCloud

Synchronisation iCloud

If you install Link2Mat on multiple devices signed into the same iCloud account, your changes sync between those devices. Only changes are transferred.

同じ iCloud アカウントで複数端末にインストールした場合、変更内容が端末間で同期されます。 変更分のみが送受信されます。

같은 iCloud 계정으로 여러 기기에 Link2Mat을 설치하면 변경 사항이 기기 간에 동기화됩니다. 변경분만 전송됩니다.

Se você instalar o Link2Mat em vários dispositivos com a mesma conta iCloud, as mudanças serão sincronizadas entre eles. Apenas as alterações são transferidas.

Si instalas Link2Mat en varios dispositivos con la misma cuenta de iCloud, los cambios se sincronizan entre ellos. Solo se transfieren los cambios.

Si vous installez Link2Mat sur plusieurs appareils avec le même compte iCloud, les changements sont synchronisés entre eux. Seules les modifications sont transférées.

Optional Telemetry

共有データ（任意）

선택적 데이터 공유

Telemetria opcional

Telemetría opcional

Télémétrie optionnelle

If you opt in to data sharing, Link2Mat may send anonymous usage statistics (such as tag usage counts and viewing counts) and optional profile fields you provide (such as country/region). This is used to improve features and show aggregated insights inside the app.

You can turn data sharing on or off in Tag → Usage. Share profile and Share tag stats do not include names, email addresses, or contact details.

データ共有に同意した場合、Link2Mat は匿名の利用統計（例: タグの使用回数や閲覧回数）と、 任意で入力したプロフィール情報（例: 国/地域）を送信することがあります。これは機能改善と アプリ内の集計表示のために利用します。

データ共有は タグ → 利用状況 でオン/オフを切り替えられます。プロフィール共有・タグ統計を共有の いずれにも、氏名・メールアドレス・その他の連絡先は含みません。

데이터 공유에 동의하면 Link2Mat이 익명 사용 통계(예: 태그 사용 횟수, 조회 횟수)와 선택적으로 입력한 프로필 정보(예: 국가/지역)를 전송할 수 있습니다. 이는 기능 개선과 앱 내 집계 표시를 위해 사용됩니다.

데이터 공유는 태그 → 사용 현황에서 켜거나 끌 수 있습니다. 프로필 공유・태그 통계 공유 어느 쪽에도 이름, 이메일, 연락처는 포함되지 않습니다.

Se você optar por compartilhar dados, o Link2Mat pode enviar estatísticas anônimas de uso (por exemplo, contagens de tags e visualizações) e campos opcionais de perfil fornecidos (como país/região). Isso é usado para melhorar recursos e mostrar estatísticas agregadas no app.

Você pode ativar ou desativar o compartilhamento em Tag → Uso. Compartilhar perfil e Compartilhar estatísticas de tags não incluem nome, e‑mail nem dados de contato.

Si aceptas compartir datos, Link2Mat puede enviar estadísticas anónimas de uso (por ejemplo, recuentos de etiquetas y visualizaciones) y campos de perfil opcionales que proporciones (como país/región). Esto se usa para mejorar funciones y mostrar estadísticas agregadas en la app.

Puedes activar o desactivar el intercambio en Tag → Uso. Compartir perfil y Compartir estadísticas de tags no incluyen nombres, correos electrónicos ni datos de contacto.

Si vous acceptez le partage de données, Link2Mat peut envoyer des statistiques d’usage anonymes (par exemple, le nombre de tags et de vues) ainsi que des champs de profil facultatifs (comme le pays/région). Cela sert à améliorer les fonctionnalités et à afficher des statistiques agrégées dans l’app.

Vous pouvez activer ou désactiver le partage dans Tag → Utilisation. Partager le profil et Partager les statistiques de tags n’incluent ni noms, ni adresses e‑mail, ni coordonnées.

Servicios de terceros

Services tiers

When you add a link, Link2Mat may fetch page metadata (title/description/thumbnail) from the source website to improve display.

リンク追加時に、表示改善のために元サイトからメタデータ（タイトル/説明/サムネイル）を取得する場合があります。

링크를 추가할 때 표시를 개선하기 위해 원본 사이트에서 메타데이터(제목/설명/썸네일)를 가져올 수 있습니다.

Ao adicionar um link, o Link2Mat pode buscar metadados (título/descrição/miniatura) no site de origem para melhorar a exibição.

Al añadir un enlace, Link2Mat puede obtener metadatos (título/descripción/miniatura) del sitio de origen para mejorar la visualización.

Lors de l’ajout d’un lien, Link2Mat peut récupérer des métadonnées (titre/description/vignette) depuis le site source afin d’améliorer l’affichage.
