Link2Mat Privacy Policy / プライバシーポリシー

Last updated: 2026-01-25

更新日: 2026-01-25

Overview / 概要

Link2Mat helps you save links and clips for Jiu‑Jitsu learning. This policy explains how your data is handled.

Link2Mat は柔術学習のためのリンクやクリップを保存するアプリです。 本ポリシーではデータの取り扱いについて説明します。

Data We Store / 保存されるデータ

  • Saved links and metadata you add
  • Titles, notes, tags, and clips you create
  • 追加したリンクやメタデータ
  • 作成したタイトル・メモ・タグ・クリップ

Data Usage / 利用目的

We use this data to provide search, tagging, and clip features.

検索・タグ付け・クリップ機能の提供のために利用します。

Third‑Party Services / 外部サービス

When you add a link, Link2Mat may fetch page metadata (title/description/thumbnail) from the source website to improve display.

リンク追加時に、表示改善のために元サイトからメタデータ（タイトル/説明/サムネイル）を取得する場合があります。
