Tag Help / タグヘルプ

Last updated: 2026-01-28

更新日: 2026-01-28

Tag は「ポジション × シナリオ × セットアップ × アクション」で整理する考え方です。 例: どんなポジション（ハーフガード）で、どんなシナリオ（ボトム）で、 どんなセットアップ（手首を取れた）時に、どんなアクション（キムラロック）をするか。

シナリオは Gi / No-Gi、トップ / ボトム、試合 / 練習 / 教則、攻め / 守り / エスケープ などの文脈を表します。ソースは動画の媒体やルールセット（IBJJF / ADCC / Rules Unknown） を表し、試合の動画ではシナリオの「試合」と組み合わせて使います。

タグによってテクニックの言語化に慣れ、仲間や先生とのコミュニケーションの助けになり、 柔術脳の発達に貢献できればと考えています。
