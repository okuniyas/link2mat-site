Versión 2.5

Version 2.5

  • 2.5.0
    • Reference sample: a Triforce Online sample list is included on first launch so you can explore tagging with real techniques (re-import anytime from Add → Import after you remove it).

      • Add (URL, Photos, and Import tabs), Retention study, and Quick Tag — tap ⓘ for short tips and links to help videos and FAQs on the support site.
    • Quick Tag:
      • One leg and Both legs tags in Lower body; body-part and approach groupings improve when you update the tag dictionary online.
  • 2.5.0
    • 参考サンプル: 初回起動時に Triforce Online のサンプルリストが入ります。実際の技とタグ付けの例を試せます（削除後は追加 → 取り込みから再読み込みできます）。

      • 追加（URL・写真・取り込み）、定着学習、クイックタグの各画面に ⓘ を追加しました。タップで短い説明と、サポートサイトの動画・FAQ へつながります。

      • 下半身に片足・両足を追加。オンラインでタグ辞書を更新すると、部位やアプローチのグループ分けが改善されます。

    • 참고 샘플: 첫 실행 시 Triforce Online 샘플 리스트가 포함됩니다. 실제 기술로 태그를 체험할 수 있습니다(삭제 후에는 추가 → 가져오기에서 다시 불러올 수 있습니다).
    • 앱 내 도움말(ⓘ):
      • 추가(URL·사진·가져오기), 복습 학습, 퀵 태그 화면에 ⓘ를 추가했습니다. 탭하면 짧은 안내와 지원 사이트의 동영상·FAQ로 이동합니다.
    • 퀵 태그:
      • 하체에 한쪽 다리·양쪽 다리 태그 추가. 온라인에서 태그 사전을 업데이트하면 신체 부위·접근 방식 그룹이 개선됩니다.
  • 2.5.0

      • Tags Uma Perna e Ambas as Pernas em Parte inferior; agrupamentos melhoram ao atualizar o dicionário de tags online.
  • 2.5.0

Version 2.4

バージョン 2.4

버전 2.4

Versão 2.4

Versión 2.4

Version 2.4

  • 2.4.6
    • Fix: exporting a list no longer includes Quick Tag groups you removed. Importing a list package on another device also ignores removed groups from older export files.
  • 2.4.5
    • Fix: list exports and backup files saved from v2.4.2 or later can be imported again from Add → Import from file. They were incorrectly rejected as invalid.
  • 2.4.4
    • Lists and item detail:
      • Long notes scroll inside a fixed-height field. When the keyboard is open, the screen scrolls so the field stays visible. Tap ⌨︎ at the top to dismiss the keyboard.
    • Fix: Japanese input on list notes no longer loses focus after each character while you compose text.
    • Tag dictionary: added tags such as side roll, knee clamp, and climbing on your opponent with your knees; broader armbar aliases. When online, update from Tags → Dictionary.
  • 2.4.3
    • Lists:
      • Edit the list title and notes field by field on list detail (like item detail), so long notes are easier to save with the keyboard.
      • When you open an item from a long list and go back, the list keeps your scroll position.
      • In the list tree, collapse or expand parent lists to browse more easily.
    • Item and clip detail: saving one field no longer discards drafts you are still editing in other fields.
    • Photos tab: clearer confirmation before importing from your photo library (duplicates are called out and skipped).
  • 2.4.2
    • Clips & Shorts: pin individual clips alongside library items (shared pin limit).
      • Pin or unpin from clip detail. In Shorts & Clips, pinned clips stay at the top (like Library). In each item’s clip list, you can toggle “Pins first.”
      • Library pin guide lists pinned clips too; open a clip or unpin from the guide.
    • Lists: when manually reordering a list, sort all items by title in one tap.
    • Photo library stills: imported photos get titles from capture date/time and approximate area when location is embedded in the photo; list thumbnails are saved in the app. On item detail, photos show View instead of Play.
    • Tag dictionary: added knee twist (setup).
  • 2.4.1
    • Fix: if your library, lists, or tags looked empty after updating to 2.4.0, this update restores access to your saved data on the device (no backup restore needed).
  • 2.4.0
    • Fix: updating from v2.3.x no longer leaves your library, lists, and tags appearing empty (your data on the device is preserved).
    • Photo tab: import multiple photos or videos from your device library in one batch—ideal for instructionals you saved on your phone or tablet.
      • While importing, add items to an existing list, create a new list, or create a new sub-list under a parent.
      • Progress is shown during import, and multi-video imports are typically much faster than before.
      • When photos and longer videos are mixed, you can import quick items first and finish longer videos afterward.
    • Lists:
      • Create sub-lists under a parent list to organize by course, position, or theme.
      • Export a list from list detail to share with another device (includes sub-lists and lists with only a title and notes).
      • Import the file from Add URL; you can apply list membership for items already in your library without overwriting them, and optionally overwrite notes on matching lists.
    • Today’s retention study:
      • Titles now match item detail and other screens more consistently, so fewer existing entries show as “Untitled item.”
      • When an item is due today, mark it complete from item detail with Done, next / Done, prev (or Done only)—from the retention list or while browsing your library.
    • Library: more reliable thumbnails for pinned items and for Instagram, Facebook, and TikTok links; clearer layouts when you have many saved items.
    • Tag dictionary:
      • Added tags such as double guard, shin-to-shin, split squat (including transitional setup), sit-up, neck throw, and throw.
      • Expanded searchable aliases including shoulder lock and arm triangle.
    • As noted in earlier release notes, cross-device iCloud sync has ended. To move your library to another device, use Backup and Restore in Library settings.
  • 2.4.6
    • 修正: リストのエクスポートに、削除したクイックタグのグループが含まれないようにしました。別端末へ取り込むときも、削除済みタグが復活しません。
  • 2.4.5
    • 修正: v2.4.2 以降でエクスポートしたリストやバックアップ JSON を、追加画面の「ファイルから取り込み」で再度読み込めるようにしました。以前は無効なファイルとして誤って拒否されていました。
  • 2.4.4
    • リスト・アイテム詳細:
      • 長いメモは欄内でスクロールできます。キーボード表示時は画面がスクロールして入力欄が見える位置に調整されます。画面上部の⌨︎でキーボードを閉じられます。
    • 修正: リストのメモを日本語入力で編集するとき、1文字ごとにフォーカスが外れる不具合を直しました。
    • タグ辞書: サイドロール、ひざで挟む、ひざで相手に登る などを追加。アームバーの別名も拡充。オンライン時はタグ→辞書の更新から取得できます。
  • 2.4.3
    • リスト:
      • リスト詳細でタイトルとメモをフィールドごとに編集・保存できます（アイテム詳細と同様）。長いメモもキーボードで保存しやすくなりました。
      • 長いリストからアイテムを開いて戻ったとき、スクロール位置を保持します。
      • リスト一覧で親リストを折りたたみ／展開して、見やすく整理できます。

      • クリップ詳細からピン留め／解除。「ショート＆クリップ」ではライブラリ同様、ピン留めを上に表示。アイテム詳細のクリップ一覧では「ピンを上に」を切り替えられます。
      • ライブラリのピンガイドにクリップも表示され、そこから開いたり解除したりできます。

    • 写真ライブラリの静止画: 取り込み時に撮影日時と、写真に位置情報がある場合はおおよその場所をタイトルに反映。一覧用サムネをアプリ内に保存。アイテム詳細では写真は「表示」（動画は従来どおり再生）。
    • タグ辞書: ニーツイスト（セットアップ）を追加しました。
